# Stage 02 - Settings and Keymap Generation

## Goal

Generate a usable default keymap from strip orientation and LED-count assumptions.

## Build checklist

- define JSON settings schema
- define JSON keymap schema
- generate 88-key base mappings
- support left-to-right and right-to-left strip direction
- persist generated keymaps

## Success criteria

- a generated keymap file can be saved and reloaded
- mappings are monotonic in the expected direction
